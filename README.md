# Sales Prediction using Python

## Project Overview

This project predicts product sales based on advertising investments across TV, Radio, and Newspaper channels. Using Machine Learning and Linear Regression, the model analyzes advertising data and forecasts future sales with high accuracy.

## Objectives

* Analyze advertising and sales data
* Perform data cleaning and preprocessing
* Visualize relationships between features
* Build a Linear Regression model
* Predict future sales based on advertising budgets
* Evaluate model performance

## Dataset Features

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Data Analysis

* Checked dataset structure and missing values
* Removed unnecessary columns
* Performed correlation analysis
* Created a correlation heatmap
* Visualized relationships between advertising channels and sales

## Machine Learning Model

**Model Used:** Linear Regression

### Performance

* R² Score: 0.8994 (~90% Accuracy)

## Sample Prediction

Input:

* TV = 230
* Radio = 40
* Newspaper = 50

Predicted Sales:

* 20.97

## Key Insights

* TV advertising has the strongest impact on sales.
* Radio advertising has a moderate impact on sales.
* Newspaper advertising has the least influence on sales.
* The model provides reliable sales predictions for marketing decision-making.

## Conclusion

This project successfully predicts sales using advertising expenditure data. The Linear Regression model achieved approximately 90% accuracy, demonstrating strong predictive performance and providing valuable insights for business marketing strategies.
